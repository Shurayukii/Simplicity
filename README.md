## Installation

<!-- BetterDiscord -->
<details>
<summary><b>BetterDiscord (Use PTB)</b></summary>

* **Step 1:** Open **Command Prompt** / **Terminal**

* **Step 2:** Paste the below code in your terminal:

(for first time downloaders remove the && rd /s /q "Simplicity" from the command line otherwise it doesn't work)

```bash
cd AppData/Roaming/BetterDiscord/themes && rd /s /q "Simplicity" && git clone https://github.com/Shurayukii/Simplicity.git --branch Binky
```
</details>

